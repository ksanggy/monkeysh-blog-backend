# monkeysh-blog-backend
monkeysh-blog backend api server 입니다. monkeysh-front 프로젝트에서 사용 할 백앤드서버.

# Tech Spec
1. Java (1.8)
2. Spring Boot 2.6
3. MariaDB (10)
4. Spring Security
5. JWT - JJWT

# 목적
1. 개인 포트폴리오 백앤드 구축
2. Spring Security 활용 (특히 Token based authentication/authorization)
3. Front - Back 분리 환경 경험
